# Images Folder

## 📁 What Goes Here

Place your website images in this folder:

1. **Profile Picture**: `prince-profile.jpg` or `prince-profile.png`
2. **Portfolio Project Screenshots**: `project1.jpg`, `project2.jpg`, etc.
3. **Additional Images**: Any other images you want to use

## 🖼️ Profile Picture Setup

### Quick Start:

1. **Add your photo to this folder** (`d:\Projects\MyselfPrince\public\images\`)
2. **Name it**: `prince-profile.jpg` (or `.png`)
3. **That's it!** ✅ The website is already configured

### Requirements:

- **Format**: JPG or PNG
- **Recommended Size**: 400x400 pixels or larger (square ratio)
- **File Size**: Under 500KB for fast loading
- **Quality**: High resolution for professional appearance

## 📸 For Portfolio Images:

1. Add project screenshots here

   ```jsx

   ```

## 📝 File Naming Tips

- Use lowercase: `profile.jpg` (not `Profile.JPG`)
- No spaces: `project-screenshot.jpg` (not `project screenshot.jpg`)
- Be descriptive: `crypto-dashboard.jpg` (not `img1.jpg`)

## ✅ Recommended Specs

- **Format**: JPG or PNG
- **Profile**: 400x400px to 800x800px (square)
- **Portfolio**: 800x600px or 1200x900px (landscape)
- **File Size**: Under 500KB each
